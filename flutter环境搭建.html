Flutter 环境配置

Windows 和 Linux 操作系统，只能搭建 Android 开发环境；

如果需要同时搭建 Android 和 IOS 开发环境，请选择 MacOS。

Windows 下的 Flutter 环境配置

安装 Git 环境

联网下载最新的 Git 安装包，直接安装到计算机中即可。

安装 Flutter

  1. 通过以下命令，clone 最新的 Flutter 仓储到本地：

     git clone https://github.com/flutter/flutter.git

  2. 将 clone 的 Flutter 仓储中 bin 目录的完整路径，添加到系统环境变量 path 中。

安装 JDK

安装 JDK 成功后，按照如下步骤配置系统环境变量：

  1. 新建系统环境变量 JAVA_HOME ，值为 jdk 的安装目录

  2. 修改系统环境变量 path ，新增如下两个变量：

    • %JAVA_HOME%\bin

    • %JAVA_HOME%\jre\bin

  3. 新建系统环境变量 CLASSPATH ，值为：

    • .;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;

  4. 保存上述所有的操作，同时退出系统环境变量配置窗口，打开任意终端并输入 javac，如果能出现 javac 的命令选项，就表示配置成功！

安装 Android SDK

安装 Android SDK 成功后，按照如下步骤配置系统环境变量：

  1. 新建系统环境变量 ANDROID_HOME ，值为 android SDK Manager 的安装路径

  2. 修改系统环境变量 path ，新增如下两个变量：

    • %ANDROID_HOME%\tools

    • %ANDROID_HOME%\platform-tools

安装夜神模拟器

  1. 直接安装最新版的夜神模拟器；

  2. 将安卓 SDK 安装目录中，platform-tools 目录下的 adb.exe 拷贝一份，替换掉 Nox\bin 目录下夜神模拟器自带的 adb.exe；

  3. 将安卓 SDK 安装目录中，platform-tools 目录下的 adb.exe 拷贝一份，重命名为 nox_adb.exe 后，替换掉 Nox\bin 目录下夜神模拟器自带的 nox_adb.exe；

  4. 打开夜神多开器，新建 安卓5.1.1 模拟器；

安装 VS Code

  1. 直接安装最新版本的 VS Code；

  2. 在扩展中，搜索 flutter 和 dart 插件并安装；

  3. 创建项目并运行到夜神模拟器中查看效果。
